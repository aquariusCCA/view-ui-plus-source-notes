# View UI Plus 源碼閱讀筆記

這份筆記目錄是為了系統性閱讀 View UI Plus 源碼而設計，目標不是逐行翻譯源碼，而是沉澱成可回查、可複用、可仿寫的前端元件庫知識庫。

## 使用方式

建議使用 Obsidian、VS Code 或 GitHub 管理。

推薦閱讀順序：

1. 先讀 `00_總覽與讀碼策略`
2. 再讀 `01_根目錄與工程設定`、`02_build_打包腳本`
3. 接著讀 `03_src_入口與插件機制`
4. 然後讀 `12_src-directives_自訂指令`、`13_src-utils_工具函式`、`14_src-mixins_混入與共用邏輯`
5. 再挑核心元件深讀：Form、Input、Select、Table、Modal、Message、Upload、Tree、Cascader、DatePicker
6. 最後整理 `22_高階專題` 與 `23_仿寫與重構`

## 筆記原則

每篇筆記盡量回答以下問題：

- 這段源碼解決什麼問題？
- 它在整個元件庫中扮演什麼角色？
- 它用了哪些 Vue 觀念？
- 有哪些可帶回公司專案的設計？
- 如果自己仿寫，會如何簡化？
- 哪些問題值得記到回查庫？
