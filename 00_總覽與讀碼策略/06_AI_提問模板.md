# AI 提問模板

## 1. 解釋單一檔案

請幫我解釋以下 View UI Plus 源碼檔案：
- 它的用途是什麼？
- 它在整個元件庫中扮演什麼角色？
- 核心流程是什麼？
- 哪些地方適合做成筆記？
- 哪些設計可以帶回公司專案？

## 2. 解釋單一元件

請幫我按照以下格式分析這個元件：
1. 元件定位
2. Props
3. Emits
4. Slots
5. 狀態設計
6. 事件流程
7. 樣式設計
8. 與其他元件的關係
9. 可學走的設計
10. 簡化仿寫版

## 3. 解釋看不懂的程式碼

請不要只翻譯程式碼，請說明：
- 為什麼要這樣寫？
- 不這樣寫會發生什麼問題？
- 它處理了哪些邊界情況？
- 是否有更現代的 Vue 3 寫法？

## 4. 產生筆記

請根據以下源碼，整理成 Markdown 筆記，包含：
- 用途
- 核心流程
- 關鍵資料結構
- 設計思想
- 可仿寫版本
- 回查問題
